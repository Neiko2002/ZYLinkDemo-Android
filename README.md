# ZYLink
ZYLink 是智云移动团队用于控制稳定器的SDK需要申请证书才能使用，此项目用于演示ZYLink的基础使用。
项目中不含证书，需要证书请联系dev@zhiyun-tech.com获取

## 下载

Gradle:
```gradle

  repositories {
    maven{ url 'https://dl.bintray.com/zhiyunteam/maven/' }
  }

  dependencies {
    implementation 'com.zhiyun:zylink:1.3.0'
  }
```
